# Rick and Morty Episode Listing

Este projeto foi desenvolvido utilizando ReactJS, TypeScript e ShadcnUI com Tailwind CSS.

## Funcionalidades

- Listagem de episódios utilizando a API [Rick and Morty](https://rickandmortyapi.com/) juntamente com GraphQL.
- Funcionalidades de paginação.
- Filtragem de dados.
- Detalhamento de episódios.

## Como Rodar o Projeto

Para rodar o projeto em modo de desenvolvimento, siga os passos abaixo:

1. Certifique-se de ter o Yarn instalado.
2. Abra o terminal na pasta do projeto.
3. Execute os seguintes comandos:

```bash
yarn
yarn dev
